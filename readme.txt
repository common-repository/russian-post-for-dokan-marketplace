Фулфилмент от Почты России для маркетплейса на базе Dokan.

== Description ==
Услуга фулфимента для маркетплейса на базе Dokan заключается в организации быстрой и качественной доставки товаров от продавца к покупателю. Она помогает экономить время предпринимателям, а потребителям – получить покупку в надлежащем виде в установленные сроки.

= Возможности для покупателей =
* Выбор удобного пункта для получения заказа,
* Оформление доставки до двери,
* Расчет стоимости доставки в режиме реального времени,
* Отслеживание статуса заказа

= Возможности для продавцов =
* Отправка товаров с фулфилмент-склада почтой или курьером,
* Получение сгенерированных бланков для отправки заказов,
* Отслеживание статуса отправки,
* Автоматическое изменение статуса заказа в WooCommerce при получении посылки



== Installation ==
1. Загрузите архив с плагином через меню "Плагины – Добавить новый" или разархивируйте плагин в папку "/wp-content/plugins/",
2. Активируйте плагин через меню "Плагины" вашего сайта,
3. Перейдите в меню "Woocommerce – Настройки – Доставка", вкладка "Почта России" и заполните настройки плагина (токен, адрес API, логин/пароль от API и другие). Сохраните изменения. Повторите действия на вкладке "Курьер Почта России",
4. На вкладке "Зоны доставки" создайте зону доставки и добавьте в нее методы "Почта России" и "Курьер Почта России"


== Frequently Asked Questions ==
= Как настроить плагин, где взять необходимые ключи? =
В спецификации Почты России можно найти подробную информацию по получению ключей и токенов https://otpravka.pochta.ru/specification#/main.

= Где получить токен, логин и пароль API отправки посылок? =
Чтобы получить токен, логин и пароль от API отправки посылок, нужно зарегистрироваться в онлайн-сервисе https://otpravka.pochta.ru/ и заключить договор или указать реквизиты действующего договора. После этого данные придут на email, указанный при регистрации. Их также можно найти в личном кабинете пользователя. Подробнее (https://otpravka.pochta.ru/specification#/authorization-token)

= Где получить логин и пароль API отслеживания посылок? =
Параметры доступа к API отслеживания посылок можно получить на странице https://tracking.pochta.ru/access-settings. Предварительно нужно зарегистрироваться в сервисе https://tracking.pochta.ru/ и подключить доступ к API. Подробная информация https://tracking.pochta.ru/specification

Если вы пользуетесь услугами фулфилмента, то на странице настройки плагина необходимо заполнить нужную информацию с адресом 3pl склада, почтового отделения для склада и контактными данными ответственного лица для связи с курьером ПЭК.
